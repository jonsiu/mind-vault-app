## AI UI patterns for desktop apps

- **Chat Interface**: Implement chat UI for conversational AI
- **Streaming Display**: Display streaming AI responses token by token
- **Regenerate**: Allow regenerating AI responses
- **Edit Prompts**: Allow editing previous prompts to retry
- **Copy/Export**: Easy copy or export of AI-generated content
- **Side Panel**: Use side panel for AI assistant without disrupting workflow
- **Inline AI**: Embed AI features inline where relevant
- **Context Menu**: Add AI options to context menus
- **Keyboard Shortcuts**: Provide keyboard shortcuts for AI features
- **Voice Input**: Support voice input for prompts
- **Loading States**: Show appropriate loading states during inference
- **Typing Indicators**: Use typing indicators for streaming responses
- **AI Indicators**: Clearly mark AI-generated content
- **Feedback**: Allow users to provide feedback on AI outputs
